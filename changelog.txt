# Changelog - WooTag AI Generator

## [1.3.1] - 2025-12-30
### Fixed
- Corrección de errores de compilación y referencias indefinidas en `Controls.tsx` (funciones de búsqueda e importaciones).
- Se corrigieron imports faltantes (Iconos de Lucide React).
- Corrección de `TypeError` en `Controls.tsx` cuando se carga la app en modo invitado (sin configuración de WooCommerce).
- Se añadió la definición de tipo faltante para `showThousandsSeparator`.

## [1.3.0] - 2024-12-30
### Added
- Nueva característica: **Leyenda de Precio Configurable**. Permite agregar un texto libre debajo del precio (ej: "IVA Incluido").
- Selector de color independiente para la "Leyenda de Precio".
- Selector de tamaño de fuente independiente para "Leyenda de Precio" y "Etiqueta P. Especial".
- Nueva característica: **Separador de Miles**. Interruptor en Ajustes para activar/desactivar el formato de miles (ej: 1.000 vs 1000).
- Selector de color independiente para la "Etiqueta de Precio Especial" (ej: Mayorista).

### Fixed
- Corrección en el Código QR: Se eliminó el prefijo "SKU:" del valor escaneado, ahora muestra solo el código limpio.
- Corrección de UI: Se aumentó el ancho del menú lateral en resoluciones medias y grandes para evitar solapamientos.
- Se restauró el selector de **Tamaño QR** que había desaparecido de la interfaz.
- Normalización del nombre de archivo `README.md` para evitar conflictos de git en Windows.

## [1.2.0] - 2024-05-24
### Added
- Nuevo archivo `changelog.txt` para seguimiento de versiones.
- Nuevo archivo `readme.md` profesional para documentación en GitHub.
- Funcionalidad de "Recordar credenciales" en la pantalla de inicio de sesión mediante localStorage.

### Fixed
- Se corrigió la falta de contraste en elementos de formulario (selects y textboxes) mejorando la legibilidad.
- Se optimizó el disparador de impresión (`window.print()`) para asegurar compatibilidad en todos los navegadores.
- Ajuste de estilos CSS en `index.html` para forzar el tamaño A4 real al generar PDFs.

## [1.1.0] - 2024-05-23
### Added
- Sistema de "Perfiles de Diseño": Permite guardar y cargar diferentes configuraciones de etiquetas.
- Integración con Gemini AI (gemini-3-flash-preview) para optimización de descripciones de productos.
- Soporte para múltiples páginas en la generación de PDF/Impresión.

### Fixed
- Error de escala en la impresión: Ahora el diseño se imprime a tamaño 1:1 independientemente del zoom de pantalla.
- Visibilidad de bordes de corte en el modo de impresión.

## [1.0.0] - 2024-05-22
### Added
- Lanzamiento inicial de la aplicación.
- Conexión con la API de WooCommerce.
- Generador de códigos QR dinámicos por SKU.
- Panel de control lateral para personalización de colores, fuentes y layout.
